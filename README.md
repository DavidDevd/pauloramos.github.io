# Portfolio PHOENIX

Professional portfolio foundation for Paulo Ramos.

## Stack

- Next.js 15
- React 19
- TypeScript
- TailwindCSS
- shadcn/ui conventions
- Framer Motion
- Lucide React
- ESLint
- Prettier

## Scripts

```bash
npm install
npm run dev
npm run build
npm run lint
npm run typecheck
```
